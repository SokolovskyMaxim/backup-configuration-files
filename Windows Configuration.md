# Windows Configuration

## Quick Start 

### Initial preparation

- Update Windows
- [[Windows Pro activation]] 
- Create a directory structure on the disk `DAL1_data DAL2_data temp`
```
iwr -useb https://git.io/debloat|iex
```
- Update applications by using Microsoft Store
